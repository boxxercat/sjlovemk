# 프로그램개요서.md  
💝 **Valentine Mini Web Project – “민경아 나의 발렌타인이 되어줄래?”** 💝

---

## 1. 프로젝트 개요

본 프로젝트는 **발렌타인 데이**를 맞아 여자친구에게 서프라이즈로 전달하기 위한  
**단일 웹페이지(Single Page)** 기반의 로맨틱 인터랙션 웹 프로젝트이다.

단순한 정보 전달이 아닌,  
👉 **“선택하는 순간까지 두근거리게 만드는 경험”**  
👉 **귀엽고 장난기 있는 인터랙션**  
👉 **결국은 ‘좋아’를 누를 수밖에 없는 운명적 UX**  

를 목표로 기획되었다.

웹을 잘 모르는 사용자도 직관적으로 즐길 수 있으며,  
모바일·데스크톱 어디서든 자연스럽게 동작하도록 구현한다.

---

## 2. 사용 기술 스택

| 구분 | 기술 |
|---|---|
| 마크업 | HTML5 |
| 스타일 | CSS3 (Flexbox, Animation, Transition) |
| 스크립트 | Vanilla JavaScript (ES6) |
| 기타 | Responsive Web Design (Media Query) |

> 💡 별도의 프레임워크 없이 **가볍고 빠르게 로딩되는 순수 웹**을 지향한다.

---

## 3. 화면 구성 설명

### 3.1 메인 화면 구성

- **중앙 질문 문구**
  - 문구:  
    > **“민경아 나의 발렌타인이 되어줄래?”**
  - 화면 중앙 정렬
  - 부드러운 페이드인 또는 살짝 튀어 오르는 애니메이션 적용

- **선택 버튼 영역**
  - 버튼 2개:
    - 💖 `좋아`
    - 🙅‍♀️ `싫어`
  - 처음에는 나란히 배치
  - 버튼 크기 및 색상으로 감정 대비 표현

---

## 4. 주요 기능 설명

### 4.1 질문 표시 기능

- 페이지 로드 시 메인 질문 문구가 자연스럽게 등장
- 시선 집중을 위해:
  - 약간의 딜레이 후 등장
  - 핑크 계열 컬러 + 둥근 폰트 사용

---

### 4.2 버튼 인터랙션 기능

#### 1️⃣ "싫어" 버튼 회피 로직

- 사용자가 **"싫어" 버튼 위로 커서를 이동**하는 순간:
  - 버튼이 화면 내 랜덤 위치로 이동
  - `position: absolute` 기반으로 이동 처리
  - 화면 밖으로 벗어나지 않도록 좌표 제한

- 이 동작은 **무한 반복**
  - 사실상 클릭 불가능 😆
  - 귀엽고 장난스러운 인상 연출

---

#### 2️⃣ "좋아" 버튼 성장 애니메이션

- "싫어" 버튼에 마우스가 접근할수록:
  - `좋아` 버튼이 점점 커짐
  - scale 애니메이션 적용
  - 최대 크기 제한 설정 (과도한 확대 방지)

> 💘 UX 의도:  
> _“도망치는 싫어 vs 점점 커지는 좋아” → 자연스럽게 선택 유도_

---

### 4.3 "좋아" 버튼 클릭 이벤트

- 클릭 시 다음 이벤트 실행:

1. **화면 중앙 텍스트 출력**
   - 문구:  
     > 🎉 **“끼얏호우~”** 🎉
   - 커다랗고 귀여운 스타일
   - Bounce 또는 Pop 애니메이션

2. **축하 애니메이션 연출**
   - 하트 이펙트가 화면에 떠오름
   - 페이드인 / 플로팅 하트 / 반짝임 효과
   - 배경 색상이 살짝 더 밝은 핑크 톤으로 전환

---

## 5. 사용자 인터랙션 흐름

1. 페이지 접속
2. 질문 문구 등장 → 기대감 형성
3. 두 개의 버튼 인식
4. "싫어" 버튼에 마우스 접근
   - 버튼 도망
   - "좋아" 버튼 성장
5. 웃음 + 당황 + 호기심
6. 결국 "좋아" 클릭
7. 🎊 축하 메시지 & 애니메이션
8. 발렌타인 성공 💕

---

## 6. 애니메이션 및 UI/UX 포인트

- **전체 톤**
  - 핑크, 파스텔, 하트 중심
  - 둥근 버튼 & 부드러운 곡선

- **애니메이션 키워드**
  - `ease-in-out`
  - `transform: scale`
  - `opacity transition`
  - Floating Heart Effect

- **UX 설계 포인트**
  - 스트레스 없는 인터랙션
  - 실패 없는 선택 구조
  - 장난스럽지만 기분 좋은 연출
